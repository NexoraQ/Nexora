## ✨ Возможности

<details open>
<summary><b>💬 Общение</b></summary>

- 🔐 **E2EE** — сквозное шифрование в чатах (AES-GCM, ключ из пароля комнаты).
- 💬 **Личные и групповые чаты** — переписка в реальном времени.
- 📢 **Группы и каналы** — описание, владелец, участники, авто-вступление.
- 🔐 **Секретные чаты** — E2EE + таймер самоуничтожения сообщений (10с / 1 мин / 1 час / 1 день).
- 📎 **Фото, видео и файлы** — отправка в один тап.
- ✍️ **Статус «печатает…»** и индикаторы активности.
- 🌍 **Мультиязычность** — 10 языков (ru, en, uk, de, es, fr, zh, tr, ar, pt).
</details>

<details open>
<summary><b>🎨 Персонализация</b></summary>

- 👤 **Профиль как в Telegram** — `username`, имя, фамилия, био, дата рождения, язык, DC (1–5).
- 🎨 **Фон чата** — тёмные темы + своё фото из галереи.
- 🆔 **Уникальный ID** — у каждого свой `@id...` для поиска и подарков.
</details>

<details open>
<summary><b>🎁 Подарки и боты</b></summary>

- 🎁 **3D-подарки** — карточки с эффектами, магазин, аукционы, коллекция, валюта **Stars**.
- 🤖 **Боты** — создание, токен, авто-ответы на команды (`/start`) и упоминания.
</details>

<details open>
<summary><b>🔑 Вход и аккаунт</b></summary>

- 📧 **Регистрация по email** + код подтверждения (код приходит на почту).
- 🔑 **OAuth** — вход через GitHub / Google / Yandex.
- 🧹 **Автоудаление** неактивных аккаунтов (период 6 месяцев).
- 🛡 **Панель администратора** (`/dev`) — баннеры, режим ТО, баны, значки, уведомления.
</details>

## 🚀 Быстрый старт

1. Скачай сборку для своей платформы в разделе [Releases](../../releases).
2. Запусти приложение → введи email → получи код → введи его.
3. Общайся! Твой ID показан внизу чата — дай его друзьям для подарков.

> 💡 Вход также доступен через GitHub, Google и Yandex (если включено в сборке).

## 🔐 Безопасность

- Rate-limit регистрации и проверки кода по IP.
- Журнал безопасности.
- Права администратора проверяются **только на сервере**.
- E2EE в секретных чатах (AES-GCM, ключ из пароля комнаты).
- Автоудаление неактивных аккаунтов.
