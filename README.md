## 🚀 Как пользоваться

1. Скачай приложение для своей платформы и запусти его.
2. Введи **email** → получи **код** на почту → введи его и своё имя.
3. Открывай чаты и каналы, общайся! Твой **ID** показан внизу списка чатов — дай его друзьям, чтобы они могли отправлять тебе подарки.
4. Хочешь приватность? Включи **E2EE** и договорись о пароле комнаты с собеседником.

## 🔐 Безопасность

- Сквозное шифрование сообщений в клиенте (AES-GCM).
- Защита регистрации и входа от спама и подбора (ограничение запросов по IP).
- Все административные действия (баны, настройки) выполняются и проверяются на сервере.
