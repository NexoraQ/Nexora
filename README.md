## 💡 Что такое Nexora?

Nexora — это бесплатный мессенджер, созданный как безопасная и удобная альтернатива
привычным приложениям для общения. Вся важная логика (безопасность, блокировки,
настройки) работает на сервере, а исходный код проекта приватен — здесь вы скачиваете
готовые, проверенные сборки.

---

## ✨ Что можно делать в Nexora

- 🔐 **E2EE** — включи шифрование в чате (AES-GCM). Только вы и собеседник видите сообщения.
- 💬 **Чаты** — личные и групповые переписки в реальном времени.
- 📢 **Каналы** — создавай каналы и транслируй контент подписчикам.
- 📎 **Фото, видео и файлы** — отправляй в один клик через кнопку 📎 (работает и с обычными файлами).
- 🎤 **Голосовые сообщения** и ⭕ **видео-кружки** (как в Telegram).
- ✍️ **Статус «печатает…»** — видно, когда собеседник набирает текст.
- 📞 **Звонки** — аудио/видео WebRTC прямо из приложения.
- 🎁 **Подарки** — отправляй друзьям подарки по их ID.
- 🔔 **Уведомления** — сообщения приходят даже при закрытой вкладке.
- 🆔 **ID аккаунта** — у каждого свой `@id...` для поиска и подарков.
- 🏅 **Значки** — 👑 Владелец, ⭐ VIP, 🛡️ Модератор, ✔ Проверенный (выдаёт администрация).

---

## 🚀 Как пользоваться

1. Скачай приложение для своей платформы (см. ниже) и запусти его.
2. Нажми **«Открыть Nexora»** → введи email → получи код на почту → введи его.
3. Общайся! Твой ID показан внизу чата — дай его друзьям, чтобы они могли отправлять тебе подарки.
4. Хочешь приватность? Включи **E2EE** в боковой панели чата и задай пароль комнаты — его должен знать только собеседник.
